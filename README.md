# Interhsip Take Home Task (Working video link attached)

## Setup
### Environment
**Python version -> 3.11** <br>
**Django version -> 5.2.8** <br>

### Running Locally
```bash
cd your_directory
git clone https://github.com/Aashutosh347777/Navya-Interhsip-Task.git
pip install -r requirements.txt
py manage.py runserver
```
Visit localhost:8000 <br>

### Commands used for creating the project structure
```bash
django-admin startproject expense_tracker
py manage.py startapp expenses
py manage.py runserver
```
